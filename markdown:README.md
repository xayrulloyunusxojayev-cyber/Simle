# Simple AI Project

This project aims to create a simple AI application using React for the frontend and Node.js with Express for the backend, connected to a MongoDB database.

## Setup

1. Clone the repository.
2. Run `npm install` to install all dependencies.
3. Create a `.env` file and add your MongoDB connection string as `MONGO_URI`.
4. Run `npm start` to start the server.

## Run

- The application will be available at `http://localhost:3000`.
- Ensure you have MongoDB running on your local machine or adjust the `MONGO_URI` accordingly.
